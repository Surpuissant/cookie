# Cookie Clicker Project

## Purpose
Cookie Clicker is an incremental game where players click a cookie to earn resources and buy upgrades.

## Main Features
- **Main Cookie**: Clickable element to earn cookies.
- **Upgrades**: Improve clicks or auto-clickers.
- **Persistence**: SQLite database to save user progress.
- **User Accounts**: Authentication system.

## Architecture
- **Backend**: Node.js + Express.
- **Frontend**: Vanilla JS + CSS + EJS templates.
- **Database**: SQLite (better-sqlite3).
- **CI/CD**: GitHub Actions.

## Tools
- **Unit Testing**: Vitest.
- **E2E Testing**: Playwright.
- **Linting**: ESLint + Prettier.
- **Server**: Express.
- **Templating**: EJS.
