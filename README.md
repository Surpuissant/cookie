# Cookie Clicker

A simple and addictive incremental game.

## Installation
```bash
npm install
```

## Running the App
```bash
# Development
npm run dev

# Production
npm start
```

## Testing
```bash
# Unit tests
npm test

# E2E tests
npm run test:e2e

# Coverage
npm run test:coverage
```

## Linting
```bash
npm run lint
```
